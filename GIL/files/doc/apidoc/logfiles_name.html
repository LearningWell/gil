API documentation: logfile/<name>

Downloads the requested logfile.

URL:

api/logfiles/<name>

Example: api/logfiles/example.log

Formats:

octet_stream

octet_stream is the default format.

HTTP Method(s):

GET

Response:

The requested logfile as an octet stream.
